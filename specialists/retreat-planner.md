# Retreat Planner — Procedural Skill

## Responsibility

Plans multi-day executive retreats. Researches properties suitable for 2–4 day off-sites, curates activity options for each day, and builds the retreat experience framework.

## Procedure

### Step 1: Define the Retreat Profile

From the event brief, establish:
- Duration (2, 3, or 4 days)
- Attendee count
- Budget ceiling
- Destination preferences (countries/regions)
- Must-have elements (kosher dining, activities, natural setting)
- Nice-to-have elements (spa, golf, cultural experiences)

### Step 2: Research Properties (5–8 Target)

For each destination, find 2–3 properties that fit:

**Evaluate against:**
- Boutique / intimate (under 60 keys or private buyout possible)
- Beautiful natural setting (vineyard, mountains, waterfront, forest)
- Summer-appropriate outdoor activities
- Kosher dining path (confirmed)
- Meeting space for strategy sessions
- A/V capability
- Privacy / exclusivity

**Record per property:**
| Field | What to Find |
|-------|-------------|
| Property Name | Full official name |
| Location | City, Country |
| Type | Resort / Estate / Boutique Hotel |
| Room Count | Number of keys |
| Buyout Available | Yes / No / Partial |
| Est. Cost/Night | Per person, in USD |
| Meeting Space | Yes / No + capacity |
| Natural Setting | Describe the scenery |
| Activities Available | List 3–5 options |
| Kosher Path | Confirmed caterer or restaurant |

### Step 3: Curate Activity Options by Day

For each day:
- Morning: Strategy session (2–3 hours)
- Afternoon: Curated group activity
- Evening: Group dinner (special setting)

**Activity Selection Rules:**
1. Activities must be DESTINATION-SPECIFIC
2. At least one activity per day
3. Balance active vs. contemplative options
4. No generic "team building"

### Step 4: Build the Retreat Framework

**Day 0 — Travel Day**
- Flight logistics, arrival transfer, welcome dinner

**Day 1 — Arrival + Settle**
- Morning/afternoon: Arrive, decompress
- Evening: Welcome dinner

**Day 2 — Strategy Day 1**
- Morning: Leadership session A
- Afternoon: Activity 1
- Evening: Group dinner

**Day 3 — Strategy Day 2**
- Morning: Leadership session B
- Afternoon: Activity 2
- Evening: Closing dinner

**Day 4 — Departure**
- Morning: Reflection + closing session
- Brunch / checkout, departure flights

### Step 5: Shortlist to Top 3 Properties

Select 3 properties that best fit all criteria:
1. Best fit for group size and budget
2. Confirmed kosher dining path
3. Strongest activity options
4. Best natural setting
