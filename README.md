# Corporate Events Team

A specialized AI events team for planning corporate events — venue research, kosher compliance, RSVP management, event communications, and executive retreats.

## What This Team Does

The Corporate Events Team handles the full event lifecycle:
- **Venue Research** — Find and evaluate venues matching event criteria
- **Kosher Compliance** — Verify catering infrastructure for Jewish dietary requirements
- **Event Communications** — Save-the-dates, invitations, reminders, thank-you emails
- **RSVP Management** — Microsoft Forms, tracking, follow-ups
- **Executive Retreats** — Multi-day off-site planning with activity curation
- **Post-Event Archive** — Event logging and artifact archival

## Team Structure

| Role | Purpose |
|------|---------|
| **Orchestrator** | Entry point — receives requests, dispatches specialists |
| **Venue Researcher** | Find venues matching criteria |
| **Kosher Expert** | Verify kosher catering path |
| **Event Comms Specialist** | Draft invitations, surveys, thank-you emails |
| **RSVP Manager** | Microsoft Forms RSVP tracking |
| **Documentation Specialist** | Excel trackers, PowerPoint decks |
| **Retreat Planner** | Multi-day executive off-site frameworks |
| **Event Archivist** | Log completed events |

## Usage

This is designed to run within a Claude Code environment. The orchestrator skill (`SKILL.md`) coordinates all specialist skills.

## Client Context

This team was built for FreedomCare's events operation — a national home health care company. It can be adapted for any organization with corporate events needs.
