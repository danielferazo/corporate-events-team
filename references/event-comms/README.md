# Event Communications Archive

This folder contains all drafted and sent event communications.

## Naming Convention

- `[EventName]_SaveTheDate.md`
- `[EventName]_Invitation.md`
- `[EventName]_Reminder.md`
- `[EventName]_ThankYou.md`
- `[EventName]_FeedbackSurvey.md`

## Template Structure

Each file should contain:
- **Type:** [Save-the-Date / Invitation / Reminder / Thank-You / Feedback]
- **Sent Date:** [Date sent]
- **To:** [Who received it]
- **Subject:** [Subject line]
- **Body:** [Full email text]
- **Notes:** [Any follow-up needed]
