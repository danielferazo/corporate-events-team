# FreedomCare Events Log

**Purpose:** Archive of all events — planned, in progress, and completed.
**Maintained by:** Events Team (event-archivist)

---

## How to Use This Log

- **In Progress:** Events currently being planned
- **Completed:** Events that have happened — log actuals, notes, venue feedback
- **Archived Artifacts:** Links to all deliverables created for each event

When an event completes, the event-archivist adds:
- What actually happened (vs plan)
- Attendance actuals
- Venue rating + "would we use again?"
- Budget actuals
- What worked / what didn't
- Key learnings for future events
